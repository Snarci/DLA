## Dataset

Our data includes Colon slides from Chaoyang hospital, the patch size is 512 Ã— 512. Training set contains 1111 normal, 842 serrated, 1404 adenocarcinoma, 664 adenoma samples, and testing set contains 705 normal, 321 serrated, 840 adenocarcinoma, 273 adenoma samples.
